# User Guides

## Getting Started

1. **Sign Up** → `/signup`
2. **Create Business Page** → `/businesses/create`
3. **Add Products/SKUs** → `/admin/skus`
4. **Customize Profile** → `/business/[slug]/edit`

## Admin Guide

### Managing Listings

1. Access Admin Panel: `/admin`
2. View all businesses: `/admin/businesses`
3. Manage SKUs: `/admin/skus`

### SKU Types

| Type | Use Case | Specifications |
|------|----------|-----------------|
| Food | Restaurants, cafes | cuisine, dietary options, parking |
| Accommodation | Hotels, homestays | room type, amenities, check-in |
| Automotive | Car rental, sales | brand, model, mileage |
| Healthcare | Clinics, pharmacy | specialization, emergency |
| Education | Schools, tutoring | course type, schedule |
| Beauty | Salons, spas | service category, duration |
