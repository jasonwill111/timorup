# US-002: Compact Card Components

## Goal

Reduce padding and text sizes for BusinessCard and ProductCard on mobile.

## Changes

### BusinessCard
- Mobile: `p-2`, `text-xs`
- Desktop: `p-3`, `text-sm`
- Image: `aspect-square`

### ProductCard
- Mobile: `p-2`, `text-xs`  
- Desktop: `p-3`, `text-sm`
- Image: `aspect-square`

## Acceptance Criteria

- [x] Cards have compact padding on mobile
- [x] Text sizes reduced on mobile
- [x] Images properly sized
