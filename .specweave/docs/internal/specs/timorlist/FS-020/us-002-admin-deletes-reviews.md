# US-002: Admin deletes reviews

**Feature**: [FS-020](./FEATURE.md)

**As an** admin **I want** to delete inappropriate reviews **So that** I can moderate the platform

---

## Acceptance Criteria

- [ ] **AC-US2-01**: Admin can view all reviews on `/admin/reviews` page
- [ ] **AC-US2-02**: Admin can delete any review via API `DELETE /api/admin/reviews/:id`
- [ ] **AC-US2-03**: Delete requires admin session validation
- [ ] **AC-US2-04**: Deleted review updates business rating average
