# Content Refresh - Plan

### Phase 1: About Page Refresh

**Current Problem**: Generic business directory description

**New Positioning**:
- Hero: "Timor-Leste's Local Info Hub" + 4-in-1 tagline
- Section headers: Directory | Shop | Classifieds | Community
- Stats: Real or placeholder with context

### Phase 2: Pricing Page Context

**Add introduction block before cards**:
- Brief platform overview (2 sentences)
- "Choose what's right for you"

### Phase 3: FAQ Refresh

**Restructure to 4 categories**:
1. General (What is TimorUp?)
2. For Businesses (Directory, Shop, Pricing)
3. For Community (Classifieds, Free tier)
4. For Services (NGOs, Government)

### Phase 4: Contact Page Polish

**Keep current structure, polish content**:
- More specific contact purposes
- Clear response expectations
- Professional but welcoming tone

## Rationale

Content changes are low-risk but high-impact. Clear positioning reduces bounce rate and increases conversions. 1-hour work, lasting improvement.
